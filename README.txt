========================================
 工事写真 管理アプリ（Phase 1 / PC・コア版）
========================================

■ これは何？
電気工事の工事写真を、パソコンで整理・分類し、写真帳（PDF / Excel）に
出力するためのアプリです。インストール不要・インターネット不要で動きます。

■ 起動方法
1. このフォルダ「写真管理アプリ」ごと、パソコンの好きな場所に置きます。
2. フォルダ内の「index.html」を ダブルクリック します。
   → Microsoft Edge または Google Chrome で開いてください。

  ※ Internet Explorer / Firefox では正しく動作しない場合があります。
  ※ 既定のブラウザが Edge/Chrome でない場合は、index.html を右クリック →
     「プログラムから開く」→ Edge か Chrome を選んでください。

■ 使い方（基本の流れ）
1.「＋ 新規工事」で工事（案件）を作ります。
   ・工事名は後から変更できます。「工事の選択」の各行の「名称変更」、または
     画面上部の工事名をクリックしても変更できます。
2.「＋ 写真を取り込む」で、パソコン内の写真をまとめて選びます。
   → 撮影日時・GPS（位置情報）は写真から自動で読み取ります。
   ・同じ写真（ファイル名・サイズ・撮影日時が一致）を取り込もうとすると
     「取込み済み」の確認が出ます。「重複も取り込む／取込み済みを除く／
     キャンセル」から選べます。
3. 一覧のサムネイルをクリックすると、工種・種別・撮影区分・測点・説明などを
   入力・編集できます（Esc キーで閉じられます）。
   ・詳細画面の写真をクリックすると、全画面で拡大表示します
     （マウスのホイールや ＋／－ で拡大縮小、ドラッグで移動、Esc で閉じる）。
   位置情報が無い写真は、地図や数値で手入力できます。
   ・チェックで複数選び「選択をまとめて入力」を押すと、入力した項目だけを
     まとめて設定できます（空欄の項目は変更しません）。
   ・チェックは Shift＋クリックで範囲選択できます。
   ・「日付でまとめる」にチェックすると、撮影日ごとに区切って表示され、
     見出しの「この日を選択／解除」で日付単位にまとめてチェックできます。
   ・すでにどれかのアルバムに入っている写真は「アルバム済」と表示されます。
   ・編集の途中でウィンドウを閉じても、入力内容は自動で一時保存され、
     次に同じ写真を開くと「前回の編集途中」を復元できます（保存すると消えます）。
     一時保存がある写真には「下書き」と表示されます。
4. 画面左の「絞り込み」タブ（チェック式）で目的の写真を素早く表示できます。
   ・撮影日／工種は、記入済みの値をチェックするとその写真だけ表示。
   ・撮影区分 → 工種 → 種別 → 細別 → 撮影箇所 は連動していて、上を選ぶと
     下の候補がその範囲だけに絞られます。
   ・同じ分類で複数チェック＝いずれか、違う分類のチェック＝両方、で絞り込みます。
   ・「クリア」で全部の絞り込みを解除します。
   上部の「キーワード検索」「並べ替え」「日付でまとめる」も併用できます。
   ・写真を左の分類（工種・撮影区分・種別・細別・撮影箇所）へ
     ドラッグ＆ドロップすると、その分類が写真に自動で反映されます。
     チェックで複数選んでいるときは、まとめて反映されます。
     ・下層（工種・種別・細別・撮影箇所）へ落とすと、上位（撮影区分・工種・
       種別・細別）も一緒に反映されます。上位はチェックしていなくても、
       その値を持つ既存写真から自動で推定して埋めます（1つに定まるときのみ。
       複数の値が混在していて定まらない上位は空のままです）。
   ・工種・種別・細別・撮影区分・撮影箇所は、一度入力すると次回から
     入力欄の候補（履歴）に出ます。文字を打つと絞り込み検索もできます。
     入力欄の右にある「▼」を押すと、文字が入っていても全履歴から選べます。
     ※履歴は「すべての工事で共通」です。
   ・写真は編集画面を開かなくても削除できます。サムネイル右上の「✕」で1枚、
     チェックで選んで「選択を削除」でまとめて削除（どちらも「戻る」で復元可）。
5.「写真帳を出力」で、1ページ 3枚 / 4枚 を選んで PDF か Excel に出力します。
   → 出力ファイル名の末尾には自動で「ai」が付きます。
   ・出力画面には仕上がりのプレビューが表示されます。レイアウト（3枚/4枚）を
     切り替えて確認し、「出力する」で保存します。
   ★ 出力の対象は次の3通りから選べます：
     (1) 全件 … 上部「写真帳を出力」ボタン（この工事の全写真）
     (2) 選択 … サムネイル左上のチェックで選び、「選択した写真で写真帳」
     (3) アルバム … 下記のアルバム単位・分類項目単位で出力

■ 戻る／進む（やり直し）
・上部の「↶ 戻る」「↷ 進む」で、直前の操作を取り消し／やり直しできます。
  対象：ドラッグ分類、選択をまとめて入力、写真の編集保存、写真の削除。
  （削除した写真も「戻る」で元に戻せます。）
・キーボードは Ctrl+Z＝戻る、Ctrl+Y＝進む。
・履歴はアプリを開いている間だけ有効で、工事を切り替えると消えます。

■ アルバム（写真をまとめて出力）
・サムネイルをチェックで選び「選択をアルバムに保存」→ 名前を付けて保存。
・上部「アルバム」ボタンで一覧が開き、アルバムごとに写真帳を出力できます。
・アルバムの「編集」では、
   - 写真の並び替え（↑↓、または行の「⠿」をドラッグして移動）
   - 行間の「＋見出し／＋空欄」で、任意の位置に見出し行・空欄コマを挿入
   - 各行の「削除」で行を削除
   - 写真の「編集」…その場で工種・説明などを直せます（写真自体に反映・戻るで復元）
・同じ画面の「撮影日ごと（自動）」では、撮影日ごとに自動でまとまり、
  日付単位で写真帳を出力できます（保存不要）。
・「自動グループ」では、工種／種別／撮影区分などの項目ごとに
  自動でまとめ、その単位で出力できます（保存不要）。
・撮影日ごと・自動グループの各行にある「編集」を押すと、その集まりを
  手動アルバムとして保存して編集画面が開きます（並べ替え・自由行の挿入・
  その場編集ができます）。同じ名前のアルバムが既にあればそれを開きます。

■ 別のパソコンへ渡す・バックアップする
・「書き出し」… 選択中の工事を 1つの zip ファイルにまとめます
   （写真の原本＋分類情報）。USB などで別のPCへ渡せます。
・「取り込み」… 書き出した zip を読み込み、新しい工事として復元します。

■ データの保存場所とプライバシー
・写真や位置情報は、この端末（ブラウザ）内にのみ保存され、
  外部へは一切送信されません（ローカル完結）。
・写真の「原本（無加工）」は必ずそのまま保持されます。
・地図表示のときだけインターネット（地図タイル）を使います。
  オフラインでも緯度経度は手入力・表示できます。

■ ご注意
・大量（数千枚〜）の写真を端末にため続けると容量が苦しくなる場合があります。
  完了した工事は「書き出し」でPCやドライブへ退避してください。
・写真帳の様式は一般的なレイアウトです。発注者様式に合わせる場合は別途調整します。
