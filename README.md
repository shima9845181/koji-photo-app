# 工事写真 管理アプリ（Phase 1 / PC・コア版）

電気工事の工事写真を、パソコンやスマホのブラウザで整理・分類し、
写真帳（PDF / Excel）に出力するためのアプリです。
**インストール不要・完全ローカル動作**（写真や位置情報は端末内にのみ保存され、外部へは一切送信されません）。

## 🌐 Web版（公開URL）

ブラウザで下記URLを開くとすぐ使えます（スマホ対応）:

**https://shima9845181.github.io/koji-photo-app/**

## 💻 ローカルで使う（オフライン）

1. このフォルダ一式をパソコンの好きな場所に置きます。
2. `index.html` を **Microsoft Edge** または **Google Chrome** で開きます。
   - ※ Internet Explorer / Firefox では正しく動作しない場合があります。

## ✨ 主な機能

- 📷 **写真の取り込み**：撮影日時・GPS（位置情報）を写真から自動読み取り。重複取込みの検出。
- 🗂 **分類・絞り込み**：工種／種別／細別／撮影区分／撮影箇所などで分類。ドラッグ＆ドロップ分類、まとめて入力、キーワード検索、日付でまとめる。
- 🗺 **地図表示**：位置情報を地図（Leaflet）で表示。位置が無い写真は手入力も可能。
- 📖 **写真帳の出力**：1ページ 3枚 / 4枚 レイアウトで **PDF** または **Excel** に出力。全件・選択・アルバム単位に対応。
- 📁 **アルバム**：任意の写真をまとめて保存・並べ替え・見出し挿入して出力。
- ↩️ **戻る／進む**：分類・編集・削除の取り消し／やり直し（Ctrl+Z / Ctrl+Y）。
- 💾 **書き出し／取り込み**：工事を zip にまとめて別PCへ移動・バックアップ。

## 🔒 データとプライバシー

- 写真・位置情報・分類データは、**この端末（ブラウザ）内にのみ保存**され、外部サーバーへは送信されません。
- 写真の「原本（無加工）」は常にそのまま保持されます。
- 地図タイルの表示時のみインターネットを使用します（オフラインでも緯度経度の手入力・表示は可能）。

## 🛠 技術構成

ビルド不要の純粋な静的サイト（HTML / CSS / JavaScript）。外部ライブラリは `lib/` にローカル同梱しオフラインでも動作します。

- EXIF 読み取り: exifr
- 地図: Leaflet
- PDF 出力: jsPDF
- Excel 出力: ExcelJS
- ZIP 入出力: JSZip
